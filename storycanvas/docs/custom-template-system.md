## Template Storage Structure
- Templates saved per-user (localStorage or database)
- Each template stores:
  - Display type (folder/character/location)
  - Custom icon (if location type)
  - Custom name
  - Interior node layout (nodes + connections)

---

## Usage
When user clicks a saved template from the panel, it places a new node on the canvas with the chosen exterior style, containing the saved interior layout.
